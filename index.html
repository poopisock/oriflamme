Instructions:

There are 9 groups of links, with 10 links in each. Roblox only allows 10 favorites per minute, meaning you must wait 60 seconds between the start of each group.

Your favorite will register if there is a number next to the favorite button, if there is no number and just the favorite star then it won't register.

Counter:

The counter will keep track of each time a link is pressed, if you accidentally press a link too early then the counter will still go up.
(meaning you have to decrement it if you are using it to keep track of which groups you've already done.)

Timer:

The timer restarts once a link is pressed and can only go up to 60 seconds. Once it hits 60 seconds, it will play a beautiful sound.
